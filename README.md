# Predicting-provider-category

Insurance claims dataset used to build a model to predict provider type (proprietary data not provided).  Features include:
MemberID
Provider type
Body Region
Procedure
Gender
Age
Comorbidity score
Duration of episode
Total medical costs
Total pharmacy costs

Train set (n= 25,000)
Test set (n= 10,666)

3 models tested for accuracy: Linear SVC, K Nearest Neighbor, Gaussian Naive Bayes- KNN had highest accuracy and was run
ACCURACY OF MODEL APPLIED TO TEST SET= 91.5%
